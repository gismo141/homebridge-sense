# homebridge-sense
Homebridge plugin for Hello Sense sleep tracking device

# Installation
As of now this plugin can be installed from npm: `npm install -g homebridge-sense`

# Configuration
Example:

    {
        "accessory": "Sense",
        "name": "Sense",
        "username": "something@something.com",
        "password": "password"
    }
