# homebridge-sense
Homebridge plugin for Hello Sense sleep tracking device

# Installation
This plugin is still under development. It will be published to NPM as soon as it is ready for prime time.

# Configuration
Example:

    {
        "accessory": "Sense",
        "name": "Sense",
        "username": "something@something.com",
        "password": "password"
    }

# Roadmap
